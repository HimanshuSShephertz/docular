ngModel [NgModelController#$setViewValue] ( type in module ng )

Description

Read a value from view.

This method should be called from within a DOM event handler. For example input or select directives call it.

It internally calls all formatters and if resulted value is valid, updates the model and calls all registered change listeners.

Usage

NgModelController#$setViewValue(value);

Parameters

  • value – {string} –

    Value from the view.
